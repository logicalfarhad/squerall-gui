Steps to generate mapping
Add data source

Click the Sources menu and click CSV file item. Currently the tool only supports the CSV datasource.
Map data source column to Ontology

When a valid datasource has been added successfully click Mappings menu item. Click the entity to map each column with ontology prefixes.
Generate Mapping and RDF file

After mapping the columns click save button and after that click the Mappings menu item again. Now click the Generate mappings to see the mapping in a readonly text box. Now click the Export mappings button to generate the mapping file and RDF file.
